# Analog Delta-Sigma ADC

## Overview

This repository contains the design and simulation of a transistor-level Delta-Sigma Analog-to-Digital Converter (ADC) using LTspice.

The objective of this project is to study, design, simulate, and integrate the fundamental analog building blocks required for implementing a Delta-Sigma ADC architecture.

## Project Roadmap

- Common Source Amplifier
- PMOS Fundamentals
- Differential Pair
- Current Mirrors
- Voltage Reference
- Voltage Regulator
- Comparator
- Flash ADC
- Operational Transconductance Amplifier (OTA)
- Integrator
- 1-Bit DAC
- Delta-Sigma Modulator
